# SampleELT 仕様書

## 概要

SampleELT は、Pentaho Spoon に類似したビジュアル ETL（Extract / Transform / Load）ツールです。
Oracle・MySQL データベースおよび Excel ファイルを対象に、データの抽出・変換・出力をドラッグ＆ドロップで設計・実行できます。

## アーキテクチャ

### データフロー

各ステップは以下のシグネチャで実装されます。

```csharp
Task<List<Dictionary<string, object?>>> ExecuteAsync(
    List<Dictionary<string, object?>> inputData,
    IProgress<string> progress,
    CancellationToken ct)
```

- 入力：前ステップの出力行リスト（`List<Dictionary<string, object?>>` ）
- 出力：処理後の行リスト
- Input ステップは `inputData` を無視し、ソースからデータを取得する
- Output ステップは入力データをそのまま返す（後続ステップへのパススルー）

### 実行エンジン

`ExecutionEngine` はカーンのアルゴリズム（Kahn's Algorithm）でトポロジカルソートを行い、依存順にステップを非同期実行します。

1. 入力なしのルートステップを特定
2. BFS で実行順序を決定
3. 各ステップを順に `ExecuteAsync` で実行
4. エラー発生時はログに記録して続行

### パイプライン保存形式

`.etl` 拡張子の JSON ファイルとして保存します。

```json
{
  "Name": "サンプルパイプライン",
  "Steps": [
    {
      "TypeName": "OracleInputStep",
      "Id": "...",
      "Name": "Oracle Input",
      "CanvasX": 100,
      "CanvasY": 150,
      "Settings": {
        "ConnectionString": "...",
        "SQL": "SELECT * FROM employees"
      }
    }
  ],
  "Connections": [
    { "SourceStepId": "...", "TargetStepId": "..." }
  ]
}
```

## ステップ仕様

### Input ステップ

#### Oracle Input
| 設定項目 | 説明 |
|---|---|
| ConnectionString | Oracle 接続文字列 |
| SQL | 実行する SELECT 文 |

#### MySQL Input
| 設定項目 | 説明 |
|---|---|
| Server | ホスト名 |
| Port | ポート番号（デフォルト 3306）|
| Database | データベース名 |
| Username | ユーザー名 |
| Password | パスワード |
| SQL | 実行する SELECT 文 |

#### Excel Input
| 設定項目 | 説明 |
|---|---|
| FilePath | Excel ファイルパス (.xlsx) |
| SheetName | 読み込むシート名 |
| HasHeader | 1行目をヘッダーとして扱う（デフォルト: true）|

---

### Output ステップ

#### Oracle Output / MySQL Output
| 設定項目 | 説明 |
|---|---|
| ConnectionString | 接続文字列 |
| TableName | 出力先テーブル名 |
| Mode | INSERT / UPSERT |

#### Excel Output
| 設定項目 | 説明 |
|---|---|
| FilePath | 出力 Excel ファイルパス |
| SheetName | 出力先シート名（デフォルト: Sheet1）|
| IncludeHeader | ヘッダー行を出力する（デフォルト: true）|

---

### Transform ステップ

#### Filter
| 設定項目 | 説明 |
|---|---|
| FieldName | フィルタ対象のフィールド名 |
| Operator | equals / notEquals / contains / greaterThan / lessThan / isNull / isNotNull |
| Value | 比較値（isNull / isNotNull の場合は不要）|

#### Calculation
| 設定項目 | 説明 |
|---|---|
| OutputFieldName | 計算結果を格納するフィールド名 |
| ExpressionType | add / subtract / multiply / divide / concat / constant |
| Field1 | 演算対象フィールド 1 |
| Field2 | 演算対象フィールド 2（constant 以外）|
| Constant | 固定値（ExpressionType = constant の場合）|
